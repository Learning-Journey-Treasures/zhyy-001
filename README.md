# 基于SpringBoot的医院His系统

#### 介绍
医院管理系统的开发目标主要包括: 实现医院内部的挂号、收费、药房、药库、科室、病床的信息化管理。提供记录医院内部各类经营决策汇总和明细的月报表、年报表。记录用户住院信息、医生信息、科室，药品，手术，病床信息等。药房管理，药房报缺。库房管理等
